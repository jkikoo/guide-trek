# Architecture commune du guide

Ce dossier contient les règles transversales appliquées avant les arbres spécialisés.

- `contexte.md` : détermine d’abord si l’utilisateur poursuit, revient, rejoint une échappatoire, arrive au bivouac, est installé ou démonte.
- `coherence.md` : barème, préconditions, incompatibilités et test de publication.
- `glossaire.md` : vocabulaire opérationnel de l’interface.

Aucune section spécialisée ne doit contourner ces règles.
